# ¬ — NOT

> *"To deny is to define the boundary"*  
> *"Тінь."*

---

## [∈](∈)

### Платонічна форма (λ-calculus)
Defined using Church Booleans:
```haskell
TRUE  ≡ λxy.x
FALSE ≡ λxy.y
¬     ≡ λp.p FALSE TRUE
```
If `p` is TRUE, it selects the first argument (FALSE).
If `p` is FALSE, it selects the second argument (TRUE).

### Type signature
```haskell
¬ :: Bool → Bool
```

**Church-Rosser**: Confluent.

## → Reduction Rules

### Boolean Logic
```
¬ TRUE  → FALSE
¬ FALSE → TRUE
```

### λ-reduction
```
(λp.p FALSE TRUE) TRUE
→ TRUE FALSE TRUE
→ (λxy.x) FALSE TRUE
→ FALSE
```

## 🎭 Role in Composition

NOT is essential for **control flow** and **logic**.
It allows for:
- Else branches (via inversion of condition).
- NAND/NOR construction (universal gates).

## 🔬 Докази

### Theorem 1: Double Negation
**Statement**: `¬(¬p) = p`

**Proof**:
If p=T: ¬(¬T) = ¬F = T
If p=F: ¬(¬F) = ¬T = F
∎
